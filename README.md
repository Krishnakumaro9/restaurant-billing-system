# Restaurant Billing System (Tkinter)

A desktop-based restaurant billing system developed using Python and Tkinter.
This project is created as part of MCA coursework and internship preparation.

## Features
- GUI-based billing application
- Add food items and quantity
- Automatic total bill calculation
- Simple and user-friendly interface
- Error handling for invalid inputs

## Tech Stack
- Python
- Tkinter

## How to Run
1. Install Python 3.x
2. Download or clone this repository
3. Open terminal in project folder
4. Run: python main.py

## Use Case
This system can be used by small restaurants or cafes to generate customer bills quickly.

## Future Enhancements
- GST and discount calculation
- Print bill option
- Database integration (SQLite)
